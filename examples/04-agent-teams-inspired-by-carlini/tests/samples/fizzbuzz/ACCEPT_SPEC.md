# Acceptance Criteria: FizzBuzz

1. **fizzbuzz function exists**: `from fizzbuzz import fizzbuzz` succeeds without error.
2. **fizzbuzz(15) is correct**: The result is a list of 15 strings: `["1","2","Fizz","4","Buzz","Fizz","7","8","Fizz","Buzz","11","Fizz","13","14","FizzBuzz"]`.
3. **fizzbuzz(1) edge case**: `fizzbuzz(1)` returns `["1"]`.
4. **CLI works**: Running `python fizzbuzz.py 20` exits with code 0 and prints 20 lines.
5. **CLI output correct**: The output of `python fizzbuzz.py 5` contains "Buzz" as the last line.

# BookStore — Skills & Agents Example

This project demonstrates every combination of Claude Code skills and agents.
See `README.md` for the full guide and combination matrix.

## Quick Reference

- **Skills** are in `.claude/skills/` — 8 skills covering all invocation modes and execution contexts
- **Agents** are in `.claude/agents/` — 3 agents demonstrating custom tools, skill preloading, and memory
- **Python code** is in `models/`, `api/`, `utils/`, `tests/` — a tiny BookStore domain
